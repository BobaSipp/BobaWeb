ABOUT

I build interactive 3D web experiences with Three.js, WebGL & GSAP. Every project is a canvas — boba shops, game UIs, product configurators, experimental art. Pixel art meets realtime rendering.

What started as a love for retro games turned into an obsession with pushing 3D graphics into the browser. No WebGL wrapper is too weird, no shader too expensive. If it runs at 60fps on a potato, I call it a win.
